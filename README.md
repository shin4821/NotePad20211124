# 가계부(AccountBoook)
가계부 만들기 프로젝트

* 기간: 2021.05.08 ~ 2021.05.28
* 사용언어: C++
* 개발환경: Visual Studio 2019
* 사용기술: 
  - MFC
  - MySQL
  - 직접만든 Array Library
